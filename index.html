Building Apps with NativeScript and Angular 2

Angular 2 is a popular JavaScript framework for building robust, high-quality applications that run in a web browser. NativeScript’s first-class Angular 2 support extends the Angular 2 runtime, making it possible for you to build native iOS and Android applications using the same framework—and in some cases the same code—that you use to build Angular 2 web apps.
